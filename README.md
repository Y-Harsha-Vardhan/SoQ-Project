# 📈 SoQ Trading Signal Backtester

This project is a robust **Backtesting Framework** built in Python to evaluate the performance of trading signals (BUY/SELL) on historical market data. The tool helps assess the effectiveness of trading strategies by computing key performance metrics, visualizing trade outcomes, and providing interactive analytics.

---

## 🔍 Problem Statement

You are given:
- A file containing **signals** (BUY/SELL/HOLD, optionally with Take-Profit and Stop-Loss levels)
- A **master dataset** of historical price data for an asset (e.g., BTC)

Your task is to simulate trades based on these signals and calculate:
- Net Profit
- Win Rate
- Drawdowns
- Sharpe Ratio
- Capital evolution over time
- Graphical visualizations of trades and performance

---

## 📁 Project Structure

| File | Description |
|------|-------------|
| `backtester.py` | Core engine that simulates trades, calculates metrics, and generates visualizations |
| `main.py` | Script to execute a backtest run and display results |
| `BTC_2019_2023_1d.csv` | Raw historical OHLCV data (used as master reference) |
| `updated_final_data.csv` | Trading signals generated by your model/strategy |
| `final_data.csv` | Possibly an earlier or alternative version of signal data |
| `Problem Statement.pdf` | Original assignment / project spec |
| `requirements.txt` | Python dependencies needed to run the project |
| `README.md` | This file, describing the project and how to use it |
